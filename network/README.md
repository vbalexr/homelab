# network

Network documentation for the home lab environment.

## Contents

Document the following in this directory:

- **Topology** — Physical and logical network diagrams.
- **VLANs** — VLAN assignments and descriptions.
- **IP Addressing** — Subnet allocations, static IP reservations.
- **Firewall Rules** — Inter-VLAN and WAN ingress/egress rules.
- **DNS** — Internal DNS zones and records.
- **VPN** — VPN configuration and access policies.

## File Naming Convention

Use descriptive Markdown files, e.g.:

```
network/
├── topology.md
├── vlans.md
├── ip-addressing.md
├── firewall-rules.md
└── dns.md
```
